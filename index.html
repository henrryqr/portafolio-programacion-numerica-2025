Introducción al Curso

Fundamentos y aplicaciones de los métodos numéricos

¿Qué es la Programación Numérica?

La programación numérica es una rama de las matemáticas y la computación que se enfoca en el desarrollo e implementación de algoritmos para resolver problemas matemáticos complejos mediante aproximaciones numéricas.

Algoritmos Eficientes

Desarrollo de métodos computacionales para resolver ecuaciones y sistemas matemáticos

Análisis de Error

Estudio de la precisión, estabilidad y convergencia de las soluciones numéricas

Implementación Práctica

Aplicación de métodos numéricos en Python y R para problemas del mundo real

Objetivos del Curso

  • Comprender los fundamentos teóricos de los métodos numéricos
  • Implementar algoritmos numéricos en Python y R
  • Analizar y comparar diferentes métodos de solución
  • Aplicar métodos numéricos a problemas de ingeniería
  • Desarrollar habilidades en resolución computacional de problemas

import tkinter as tk

def analizar():
    funcion = entrada.get()

    if funcion == "":
        resultado["text"] = "Por favor escribe una función"
        return
    letras = ""
    cant_letras = 0
    operaciones = 0
    for i in range(len(funcion)):
        c = funcion[i]
        if c.isalpha() and c not in letras:
            letras += c
            cant_letras += 1
        if c in "+-*/^":
            operaciones += 1
        if i < len(funcion) - 1:
            sig = funcion[i + 1]
            if (c.isdigit() and sig.isalpha()) or (c.isalpha() and sig.isalpha()):
                operaciones += 1

    resultado["text"] = "Variables: " + letras + "\\nCantidad: " + str(cant_letras) + "\\nOperaciones: " + str(operaciones)

ventana = tk.Tk()
ventana.title("Analizador")
ventana.geometry("300x250")
tk.Label(ventana, text="Ingresa una función:").pack()
entrada = tk.Entry(ventana)
entrada.pack(pady=5)
boton = tk.Button(ventana, text="Analizar", command=analizar)
boton.pack(pady=5)
resultado = tk.Label(ventana, text="", bg="lightgray", width=30, height=6, anchor="w", justify="left")
resultado.pack(pady=10)
ventana.mainloop()
